## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User CLI args -> Commander.js | Untrusted input parsed by Commander |
| Scan findings -> JSON output | Findings written to stdout as JSON |
| Scan results -> .sniff/ filesystem | Results persisted to local disk |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-06 | Information Disclosure | JSON output mode | accept | Scan results contain file paths and code snippets -- this is the tool's purpose; user controls what gets scanned |
| T-01-07 | Tampering | .sniff/last-results.json | accept | Local file written by local tool; same trust as any build artifact; user controls filesystem access |
| T-01-08 | Spoofing | --fail-on input | mitigate | Validate against severity enum; reject unknown values with Commander.js error |
</threat_model>

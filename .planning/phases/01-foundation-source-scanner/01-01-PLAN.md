   - Import Severity from core/types.ts
   - Define `severitySchema` as `z.enum(['critical', 'high', 'medium', 'low', 'info'])`
   - Define `ruleConfigSchema` as `z.union([z.literal('off'), z.literal(false), severitySchema])` -- allows disabling rules or changing severity
   - Define `sniffConfigSchema` as z.object with these fields (all optional with defaults):
     - `failOn`: `z.array(severitySchema).default(['critical', 'high'])`
     - `exclude`: `z.array(z.string()).default(DEFAULT_EXCLUDE)`
     - `include`: `z.array(z.string()).default(['**/*.{ts,tsx,js,jsx,html,css}'])`
     - `rules`: `z.record(z.string(), ruleConfigSchema).default({})`
     - `scanners`: `z.array(z.string()).default(['source'])`
   - Export inferred types: `SniffConfig` (output) and `SniffUserConfig` (input)

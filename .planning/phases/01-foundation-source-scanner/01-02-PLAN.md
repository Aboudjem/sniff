   - Import `fg` from `fast-glob`, `readFile` from `node:fs/promises`, `access` from `node:fs/promises`, `join`, `dirname`, `resolve` from `node:path`
   - Import `Scanner`, `ScanContext`, `ScanResult` from scanner types
   - Import `Finding` from core types
   - Import `allRules`, `SourceRule` from rules
   - Export class `SourceScanner implements Scanner`:
     - `name = 'source'`
     - `async scan(ctx: ScanContext): Promise<ScanResult>`:
       - `const start = performance.now()`
       - Resolve enabled rules: filter `allRules` where `ctx.config.rules[rule.id]` is NOT `'off'` and NOT `false` (per D-17)
       - Collect unique include globs from enabled rules, fall back to `ctx.config.include`
       - Call `fg(includePatterns, { cwd: ctx.rootDir, ignore: ctx.config.exclude, absolute: true })`
       - For each rule that has its own `exclude`, additionally filter matched files against that exclude using fast-glob's `isMatch` or simple glob logic
       - Process files in batches of 50 using `Promise.all`
       - For each file, call `scanFile(filePath, applicableRules, ctx.rootDir)`
       - Return `{ scanner: this.name, findings, duration: performance.now() - start }`
     - `private async scanFile(filePath, rules, rootDir): Promise<Finding[]>`:
       - Read file with `readFile(filePath, 'utf-8')`
       - Split into lines
       - For each line, for each rule:
         - If rule is `broken-import`: extract the import path from the match, resolve it relative to the file's directory, check if the target file exists (try `.ts`, `.tsx`, `.js`, `.jsx`, `/index.ts`, `/index.tsx`, `/index.js`). Only report a finding if NONE of these exist. This is the conservative approach per Pitfall 3 -- only relative imports, no aliases.
         - Otherwise: test `rule.pattern` against the line. IMPORTANT: create a new RegExp from the pattern's source+flags for each line (or use a non-global regex) to avoid stateful lastIndex issues.
         - If match found, push a Finding with relative file path, line number (1-indexed), column from match.index+1, and the trimmed line as snippet
     - `private resolveRules(config): SourceRule[]`: filter allRules based on config.rules settings
